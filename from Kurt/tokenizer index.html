The real story:

This tool provides a live, side-by-side view of how major AI providers tokenize text — great for demos and learning!
However:

  • Token counts & splits differ legitimately across providers due to unique vocabularies and algorithms.
  • Real-time API calls can fail (HTTP 500 errors are common due to rate limits, outages, or restricted endpoints).
  • Some providers (e.g. Anthropic) return only totals, never detailed tokens. Others may use approximations when live queries fail.
  • For the most accurate local testing: Use tiktoken (OpenAI), Hugging Face tokenizers (open models), or official console/API tools per provider.

Use this as a starting point to spark discussion — the "imperfections" are part of what makes tokenization such a fascinating (and unequal) part of how LLMs work!
